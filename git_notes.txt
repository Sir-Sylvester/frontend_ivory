git init:

Initializes a new Git repository in the current directory.
Creates a .git directory that tracks the version history of your project.

git log:

Shows the commit history of the repository.
Displays a list of all the commits along with details like commit hash, author, date, and commit message.

git commit -m:

Records changes in the repository with a message describing what was done.
The -m flag allows you to add a commit message directly in the command line.
